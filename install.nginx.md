## Install Nginx on Ubuntu 

### Step 1
``` shell script
# update the package database    
 sudo apt update
```
### Step 2
``` shell script
# install Nginx    
 sudo apt install nginx
```
### Step 3
``` shell script
# Start the Nginx Service    
 sudo service nginx start
```
### Step 4
``` shell script
# Check Status of the Nginx Service    
 sudo service nginx status
```
